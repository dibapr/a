# belajar-redux-toolkit

Tools yang dipakai:
1. Vite React
2. Bulma CSS
3. Axios
4. Redux Toolkit
5. React Router Dom
6. JSON server

## Cara develop:
1. clone repo ini
2. buka terminal, jalanin deployment server nya

```npm run dev```

3. buka terminal lagi, jalanin json server nya

```json-server --watch db.json --port {port berapa aja terserah}```
